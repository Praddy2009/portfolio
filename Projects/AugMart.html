AugMart

A future ready E-commerce Website with AR feature

Developer: Praddyum Verma


Description

AugMart is a prototype of an E-commerce website project which is going to be what future E-commerce websites will offer. It currently provides the feature of Augmented Reality(AR) for best user experience. As AR will be playing a major role in Industry 4.0, so I thought' of merging the shopping Experience with it.   


Currently this website has 4 commodities to choose (Noodles, Television, PhotoFrame and Jenga) which the user can choose and before buying can view it in AR.

Technology Used

  • FrontEnd (Html5, CSS3, Javascript)
  • Maya & Substance Painter (3D Assets)
  • Model Viewer
